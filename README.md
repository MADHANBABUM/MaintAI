# MaintAI

AI Powered Predictive Maintenance Platform built using FastAPI and Next.js.

---

## Tech Stack

### Backend

- FastAPI
- SQLAlchemy
- SQLite
- Pydantic

### Frontend

- Next.js
- React
- TypeScript
- Tailwind CSS

---

## Features

- Home API
- Health Check API
- Dashboard Statistics
- Create Machine
- Get All Machines
- Search Machines
- Get Machine By ID
- Update Machine
- Delete Machine

---

- app/

- public/

---

## Run Backend

cd Backend

uvicorn main:app --reload

Backend URL

http://127.0.0.1:8000

Swagger

http://127.0.0.1:8000/docs

---

## Run Frontend

cd Frontend

npm install

npm run dev

Frontend URL

http://localhost:3000
